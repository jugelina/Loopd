# Loopd

Get in the Loop, Stay in the Loop. A centralized app for clubs and organizations to post events — by students, for students.

## Features

- **Home** — Stories (click-through), upcoming events, recommended clubs
- **Hot This Week** — One week at a time, most popular events
- **Discover** — Share interests; get club recommendations
- **Club pages** — Media, events, posts; Follow to RSVP for members-only events
- **Events** — RSVP, Add to Calendar; follow the org to RSVP for members-only events
- **Notifications** — Events, RSVPs, reminders; unfollow to turn off
- **Profile** — Switch between Student and Club leader / Admin

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:5204](http://localhost:5204).

## Build

```bash
npm run build
npm run preview
```
